# 🌾 Crop Recommendation Agent

An AI-powered Crop Recommendation System built using Machine Learning and Streamlit.

## 📌 Features

- Predicts the best crop based on soil nutrients
- Uses Random Forest Machine Learning model
- Interactive Streamlit web interface
- Prediction history
- Farm management records
- Crop scheduling
- Easy-to-use dashboard

## 📂 Project Structure

```
crop-recommendation-agent/
│── app.py
│── prediction.py
│── train_model.py
│── logger.py
│── schedule.py
│── farm_records.py
│── requirements.txt
│── dataset/
│── models/
```

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## 🚀 Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Dataset

Crop Recommendation Dataset containing soil nutrients and environmental conditions.
